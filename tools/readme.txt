Dans ce dossier se trouvent quelques outils developpes pour faciliter le
dev du robot au sein du club.

Contenu du dossier :
- serial-to-tcp.py : Simple script a faire tourner sur le PC embarque qui
  fait un simple pont pour faire du debug over wifi. Quick and dirty, by Antoine
- build_doc.py : Script generant la doc et parsant les warnings de doxygen
  pour les rendre plus lisibles.
